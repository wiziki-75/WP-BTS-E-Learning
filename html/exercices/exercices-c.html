Exercice 1 : Calcul de la Surface et du Périmètre

Écrivez un programme en langage C qui calcule la surface et le périmètre d'un rectangle.

Utilisez les dimensions suivantes :

  • Longueur : 3.7 unités
  • Largeur : 7.4 unités

Le programme doit :

  1. Définir les variables nécessaires pour la longueur, la largeur, la surface et le périmètre.
  2. Attribuer les valeurs de longueur et de largeur telles que : longueur = 3.7 et largeur = 7.4.
  3. Calculer la surface du rectangle en multipliant la longueur par la largeur.
  4. Calculer le périmètre du rectangle en utilisant la formule : périmètre = 2 * (longueur + largeur).
  5. Afficher la surface et le périmètre calculés.


Exercice 2 : Calcul des Solutions d'une Équation Linéaire

Écrivez un programme en langage C qui calcule les solutions d'une équation linéaire de la forme ax + b = 0.

Utilisez les coefficients suivants :

  • a = 3
  • b = 0

Le programme doit :

  1. Définir les variables nécessaires pour les coefficients a, b et la solution x.
  2. Attribuer les valeurs a = 3 et b = 0.
  3. Vérifier si l'équation est de la forme 0x + b = 0. Si c'est le cas, imprimer "Ensemble des solutions est R" si b est égal à zéro, sinon imprimer "Ensemble des solutions est vide".
  4. Si l'équation n'est pas de cette forme, calculer la solution x en utilisant la formule : x = -b/a.
  5. Afficher la solution calculée.


Exercice 3 : Vérification de Nombre Parfait

Écrivez un programme en langage C qui vérifie si un nombre est parfait.

Utilisez les paramètres suivants :

  • n = 6

Le programme doit :

  1. Définir les variables nécessaires pour le nombre n, la somme s, et un compteur i.
  2. Attribuer la valeur 6 à n.
  3. Initialiser la variable s à zéro.
  4. Utiliser une boucle for pour itérer de 1 à n-1.
  5. Dans la boucle, vérifier si n est divisible par i. Si oui, ajouter i à s.
  6. Après la boucle, vérifier si n est égal à s. Si oui, imprimer "Votre nombre est parfait", sinon imprimer "Votre nombre n'est pas parfait".


Exercice 4 : Calcul de la Moyenne d'un Tableau

Écrivez un programme en langage C qui calcule la moyenne des éléments d'un tableau.

Utilisez le tableau suivant :

  • {1, 12, 13, 5, 6, 8, 9}

Le programme doit :

  1. Déclarer un tableau tab de taille 7 et l'initialiser avec les valeurs spécifiées.
  2. Déclarer une variable s pour la somme et une variable m pour la moyenne.
  3. Utiliser une boucle pour parcourir les éléments du tableau et calculer la somme.
  4. Calculer la moyenne en divisant la somme par la taille du tableau (7).
  5. Afficher la moyenne.


Exercice 5 : Calcul du Factoriel

Écrivez un programme en langage C qui calcule le factoriel d'un nombre donné.

Utilisez le nombre suivant :

  • n = 9

Le programme doit :

  1. Définir une fonction factoriel qui prend un entier n en argument et calcule son factoriel.
  2. Dans la fonction, déclarer des variables nécessaires pour le calcul du factoriel.
  3. Utiliser une boucle pour calculer le factoriel de n.
  4. Afficher le résultat du calcul du factoriel.
  5. Dans la fonction main, appeler la fonction factoriel avec nb = 9.
